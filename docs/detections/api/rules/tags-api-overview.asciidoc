=== Tags endpoint

Aggregates and returns all rule tags.

NOTE: Console now supports sending requests to {kib} APIs: prepend any {kib} API endpoint with `kbn:` and send the request via Console, for example, `GET kbn:/api/index_management/indices`.

==== Get tags

Aggregates and returns all unique tags from all rules.

===== Request URL

`GET <kibana host>:<port>/api/detection_engine/tags`

====== Example request

Gets tags for all rules in the {kib} default space:

[source,console]
--------------------------------------------------
GET api/detection_engine/tags
--------------------------------------------------
// KIBANA

===== Response code

`200`::
    Indicates a successful call.

====== Example response

[source,json]
--------------------------------------------------
[
  "zeek",
  "suricata",
  "windows",
  "linux",
  "network",
  "initial access",
  "remote access",
  "phishing"
]
--------------------------------------------------
