AWS CloudTrail Log Updatededit

Identifies an update to an AWS log trail setting that specifies the delivery of log files.

Rule type: query

Rule indices:

  • filebeat-*

Severity: low

Risk score: 21

Runs every: 10 minutes

Searches indices from: now-60m (Date Math format, see also Additional look-back time)

Maximum alerts per execution: 100

References:

  • https://docs.aws.amazon.com/awscloudtrail/latest/APIReference/API_UpdateTrail.html
  • https://awscli.amazonaws.com/v2/documentation/api/latest/reference/cloudtrail/update-trail.html

Tags:

  • AWS
  • Elastic

Version: 1

Added (Elastic Stack release): 7.9.0

Rule authors: Elastic

Rule license: Elastic License

Potential false positivesedit

Trail updates may be made by a system or network administrator. Verify whether the user identity, user agent, and/or hostname should be making changes in your environment. Trail updates from unfamiliar users or hosts should be investigated. If a known behavior is causing false positives, it can be excluded from the rule.

Rule queryedit

event.action:UpdateTrail and event.dataset:aws.cloudtrail and
event.provider:cloudtrail.amazonaws.com and event.outcome:success
