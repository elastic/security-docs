Use these commands to uninstall ((elastic-endpoint)) from a host **ONLY** if [uninstalling an ((agent))](((fleet-guide))/uninstall-elastic-agent.html) is unsuccessful.

<DocTabs>
    <DocTab name="macOS">
        ```shell
        cd /tmp
        cp /Library/Elastic/Endpoint/elastic-endpoint elastic-endpoint
        sudo ./elastic-endpoint uninstall
        rm elastic-endpoint
        ```
    </DocTab>
    <DocTab name="Linux">
        ```shell
        cd /tmp
        cp /opt/Elastic/Endpoint/elastic-endpoint elastic-endpoint
        sudo ./elastic-endpoint uninstall
        rm elastic-endpoint
        ```
    </DocTab>
    <DocTab name="Windows">
        ```shell
        cd %TEMP%
        copy "c:\Program Files\Elastic\Endpoint\elastic-endpoint.exe" elastic-endpoint.exe
        .\elastic-endpoint.exe uninstall
        del .\elastic-endpoint.exe
        ```
    </DocTab>
</DocTabs>
