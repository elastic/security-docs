To properly deploy ((elastic-endpoint)) without a Mobile Device Management (MDM) profile, you must manually enable additional permissions on the endpoint before ((elastic-endpoint)) can be fully functional. For more information, refer to the instructions for your macOS version:

* <DocLink slug="/serverless/security/install-endpoint-manually">Install ((elastic-endpoint)) manually on macOS Catalina through Monterey</DocLink>
* <DocLink slug="/serverless/security/deploy-elastic-endpoint-ven">Install ((elastic-endpoint)) manually on macOS Ventura and higher</DocLink>

## Minimum system requirements

| Requirement                        | Value    |
|------------------------------------|----------|
| **CPU**                            | Under 2% |
| **Disk space**                     | 1 GB     |
| **Resident set size (RSS) memory** | 500 MB   |
